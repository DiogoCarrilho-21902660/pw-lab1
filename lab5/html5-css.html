HTML5 e CSS

HTML5

HTML5 é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto por Opera Software. É a quinta versão da linguagem HTML.


Elementos HTML5
header  Indica cabeçalho
nav  indica menu de navegação com links para as páginas
main  indica conteúdo principal e único, que não se repete.
article  indica porção do documento que pode ser vista de forma independente.
section  especifica uma parte do conteúdo, usada em conjunto com etiquetas h1|h2|…
aside  indica anotações, publicidade ou outros assuntos desconexos.
footer  indica rodapé da secção (com copyright).
details  define detalhes adicionais que o utilizador pode ver ou esconder, contidos num elemento summary
meter  define uma medida escalar estática dentro de um intervalo
progress  define uma barra de progresso de uma tarefa
abbr  indica o significado de uma abreviatura em conjunto com o atributo title
address  informação sobre o contacto do autor

CSS

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web. O código CSS pode ser aplicado diretamente nas tags ou ficar contido dentro das tags "style". Também é possível, em vez de colocar a formatação dentro do documento, criar um link para um arquivo CSS que contém os estilos.


Seletores CSS
universal  Aplica o estilo a todos os elementos da página.
tipo  Usa o nome do marcador a estilizar.
classe  Associa um nome a um estilo.
id  Permite identificar univocamente um elemento do HTML
atributo  Seletores que se aplicam a atributos de elementos.
pseudo-classe  Seletor associado ao estado de um elemento.
pseudo-elemento  Regras que se aplicam a partes de elementos.

Combinadores de seletores
Seletores descendentes (espaço)  Seletores descendentes são separados por espaço
Seletores filhos (>)  Seletores filhos através do uso de >
Seletores adjacentes (~)  Seletores adjacentes através do uso de ~
Seletores imediatamente adjacentes (+)  Seletores imediatamente adjacentes através do uso de +
Agrupar os seletores (,)  Agrupar os seletores,separando-os por virgulas
